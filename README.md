# 自治体生成AIレーダー

Municipal GenAI Radar

日本の自治体における生成AIの
実証・調達・契約・本導入・定量効果・共同利用・横展開を
公開情報に基づいて追跡するデータベースです。

## 公開サイト

https://pipimanon1-cpu.github.io/municipal-genai-radar/

## 現在の収録状況

- 案件数：25件
- イベント数：16件
- 初期検証版
- 更新日：2026-07-19

上記の件数は初期収録データに基づくものであり、全国の自治体全体を代表する統計ではありません。

## 主な機能

- 案件のキーワード検索
- 都道府県、ステータス、カテゴリによる絞り込み
- 実測、推計、事前試算、目標の区別
- 案件ごとの参照元確認
- 構想、実証、本導入、効果公表などの時系列表示
- 続報未確認案件の追跡
- 初期分析レポート

## データ構成

- `data/validation/cases.csv`：案件の現在状態
- `data/validation/events.csv`：案件の時系列イベント
- `reports/2026-initial-analysis.md`：初期分析レポート

## データの考え方

- unknownは情報が存在しないことではなく、本データベースでは未確認という意味
- 続報未確認は事業終了を意味しない
- 実測、実測からの推計、事前試算、目標を区別する
- 情報は各案件のlast_checked時点のもの
- 最新情報は各案件の参照元で確認する

## データ検証

cases.csvとevents.csvの入力内容は、以下のコマンドで検証できます。

```powershell
powershell -ExecutionPolicy Bypass -File scripts/validate-data.ps1
```

## 免責事項

本データベースは公開情報を基に整理していますが、契約金額、期間、利用人数、
最新状況などが未確認の案件を含みます。掲載内容の正確性・完全性を保証する
ものではないため、利用の際は各案件の参照元で最新情報をご確認ください。
